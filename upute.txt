Dobrodosli u PHP/MYSQL aplikaciju za generiranje drop-down izbornika.

Tehnologije
UBUNTU 16.04 VM
PHP 7.0.32
MYSQL  Ver 14.14 Distrib 5.7.24
HTML 5

Upute
CREATE DATABASE dropdown;
USE dropdown;
CREATE TABLE NavMenu (id int(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY, parent_id int(6), name VARCHAR(50) NOT NULL);

Preko podatka parent_id povezujem unose i omogucujem stvaranje 'dependent' dropdown izbornika.
U varijablu name se sprema ime svakog unosa.

File conn.php uspostavlja konekciju prema Mysql bazi podataka.
File main.css je front-end styling za nastali dropdown izbornik.
File delete.php brise unose u bazi, dovodi ju u pocetno, prazno stanje.

File index.php sadrzi formu u koju se unose stavke izbornika, rucno, jedna po jedna.
File query.php preuzima post varijable, sprema ih u bazu prema pravilu "parent_id" i graficki prikazuje odnos unesenih stavki.
File dropdown.php sluzi za prikaz padajuceg izbornika s trenutnim podacima iz baze.

Ne koristim rekurziju, koristim 2 polja, parent i children, array funkcije each(), array_push() i array_pop() kako bih pokupio sve unose i razvrstao ih u liste.
